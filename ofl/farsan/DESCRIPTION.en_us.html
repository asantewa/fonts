Farsan is a single-weight typeface that supports the Gujarati and Latin scripts. Farsan is a clean, slanted and legible typeface that has great readability for titling as well as small sizes. It is useful for both on-screen and print applications.

The Farsan project is led by Pooja Saxena, a type designer based in Bangalore, India. To contribute, see github.com/anexasajoop/farsan
