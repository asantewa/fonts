The Oxygen typeface family is created as part of the KDE Project, a libre desktop for the GNU+Linux operating system. The design is optimized for the FreeType font rendering system and works well in all graphical user interfaces, desktops and devices. Oxygen Mono is the monospaced companion to the Oxygen family.

This is a web font version of Oxygen, designed to be used freely across the internet by web browsers on desktop computers, laptops and mobile devices.
