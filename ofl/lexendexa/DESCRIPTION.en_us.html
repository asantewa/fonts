Lexend is a collection of font families empirically shown to significantly improve reading-proficiency. As prescription eyeglasses achieve proficiency for persons with short-sightedness, Lexend's families were developed using Shaver-Troup Formulations.

This is the Exa family. In total there are seven sibling families, Regular, Kilo, Mega, Giga, Tera and Peta.

We will eventually release all seven families as a single variable font.

To contribute, see github.com/ThomasJockin/lexend. For more information, see thomasjockin.github.io/lexend/.
